# ImageRecognition
图像识别解决方案
# 项目说明
本仓库使用到了Emgucv和AForge，各个代码类提供的只是方案代码
要想使用，请结合场景融入自己的项目，**<font color=red size=3>此为demo，融入前请先读懂，并注意编码规范</font>**
## ImageHelper   
图片帮助类

## QrCodeHelper
二维码提取识别类，使用了Emgucv和zxing

## GetPaperFromImage
从大图中截取纸张区域，类似高拍仪拍照的图像截取，使用了Aforge类库

## WaterMarkTest
给图片添加水印，处理了包括：8位、24位、32位的图片
